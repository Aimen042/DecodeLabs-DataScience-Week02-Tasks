# Data Visualization and Predictive Modeling

## Project Overview
This project focuses on analyzing sales data through data visualization techniques and building a simple predictive model using machine learning. The objective is to communicate insights effectively and understand how different factors influence sales performance.

---

# Task 04: Data Visualization

## Goal
The goal of this task is to create visual representations of the dataset to identify trends, patterns, relationships, and key business insights.

## Visualizations Used
- Bar Chart
- Line Chart
- Histogram
- Pie Chart
- Scatter Plot
- Heatmap

## Key Insights
- Sales trends were analyzed over different years.
- Product categories with the highest sales were identified.
- Relationships between sales and profit were visualized.
- Correlation between numerical features was analyzed using a heatmap.
- Market-wise sales distribution was explored using pie charts.

## Libraries Used
- Pandas
- Matplotlib
- Seaborn

---

# Task 05: Predictive Modeling and Insights

## Goal
The goal of this task is to build a simple machine learning model to predict sales and generate meaningful business insights from the dataset.

## Machine Learning Model
### Linear Regression
A Linear Regression model was used to predict sales based on:
- Profit
- Quantity
- Discount
- Shipping Cost

## Steps Performed
1. Loaded and cleaned the dataset
2. Converted important columns into numeric format
3. Selected features and target variables
4. Split the dataset into training and testing sets
5. Trained the Linear Regression model
6. Generated predictions
7. Evaluated model performance

## Model Evaluation Metrics
- Mean Absolute Error (MAE)
- R² Score

## Key Insights
- Profit and quantity positively impact sales.
- Excessive discounts may reduce profitability.
- Shipping cost affects overall sales performance.
- Historical data can be used to estimate future sales trends.

## Libraries Used
- Pandas
- Scikit-learn
- Matplotlib
